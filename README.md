# Personal Portfolio Website

My personal portfolio website built with React, TypeScript, and Tailwind CSS.

## Features
- Responsive design
- Dark mode support
- Interactive animations
- Project showcase
- Contact form
- Particle background effects

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Vite
- React-tsparticles
- Lucide Icons
